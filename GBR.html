GBR

GameBoy => PC

UP => Up Arrow

DOWN => Down Arrow

LEFT => Left Arrow

RIGHT => Right Arrow

A => X

B => Z

Start/Fire => Enter/Return

C => Shift

D => Space

"Power Off" via Right Mouse Click
Description GBR is a development of David Winchurch's GameBoyEMU that runs on MIDP 1.0 or higher Mobile Phones.

It emulates the original Nintendo GameBoy and the GameBoy Color.

No linking and no saving, but it runs O.K. if the phone is fast enough.

The code is slightly tailored to the individual GameBoy ROM's, so a jar includes both the ROM and the code.

You should only play with these if you own the originals. All Trademarks Acknowledged.
